#

This is the lite version of [Mineprep](https://github.com/piggestpig/Unreal-Mineprep), a plugin for making Minecraft animations in Unreal Engine. It is small in size, fast to install, and open source under the GPLv3 license.


Get more information in the [full version Readme](https://github.com/piggestpig/Unreal-Mineprep/blob/main/README_English.md).  
> The lite version will be updated irregularly, and more content may be removed in the future.

> You can also do a lite installation after downloading the full version.
